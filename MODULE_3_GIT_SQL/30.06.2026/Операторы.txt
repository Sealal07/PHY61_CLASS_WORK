Поле-столбец (вертикаль)(атрибут)
Записи-строка (горизонталь)(кортеж)
Отношения (relation)-основная еденица БД, описывает
только одну сущность
Первичный (primary key)-уникальный, не пустой
Внешний (foreign key)-запись, которая ссылается на
первичный ключ другой таблицы
Типы связей:
1) 1:1
2) 1:N
3) N:N
SQL-язык структурированных запросов
1)DDL-data definition language:
CREATE - создать
ALTER - изменить 
DROP - удалить
Объекты БД: таблицы, функции, триггеры, 
процедуры, БД
2)DML - data manipulation language
INSERT INTO - вставка
UPDATE - обновление
DELETE - удаление
3)DQL - data query language
SELECT - выбрать

CRUD -  4 обязательных операции с данными

Data Types - TEXT, INT, FLOOR, BOOLEAN
